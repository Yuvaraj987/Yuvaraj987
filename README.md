## 💼 Featured Projects

### 🎙️ Mobile Devices for Impaired Persons
*Application Development, Python, Voice Recognition*
* **The Problem:** Existing assistive speech interfaces required extensive network hopping and third-party SaaS wrappers, creating high latencies up to 7 minutes for speech processing.
* **The Architecture:** Built an isolated, highly optimized Python-native processing layer wrapping speech-to-text and text-to-speech loops natively.
* **Engineering Challenge:** Multi-threaded audio buffering and CPU execution locks caused blocking latency spikes. Refactored four core processing routines to avoid unnecessary internal loop memory overhead.
* **The Impact:** Successfully dropped processing latency from **7 minutes down to a blazing 1 minute**, yielding an **85% net improvement** in assistive accessibility throughput.

### ☁️ High-Availability Serverless Application Platform
*AWS, Lambda, DynamoDB, API Gateway, S3, Python*
* **The Problem:** Traditional infrastructure setups on persistent EC2 instances generated vast cost inefficiencies during traffic lulls and demanded endless continuous OS security patching.
* **The Solution:** Ported monolithic operations entirely into an event-driven, elastic serverless state engine using microservices powered by AWS Lambda and API Gateway.
* **Performance Engineering:** Managed cold start penalties by tuning handler logic profiles and configuring selective provisioned concurrency mappings.
* **The Impact:** Realized dramatic operational overhead reductions, removing manual system configuration while significantly lowering overall infrastructure footprints compared to traditional EC2 layouts.
